# trinity-competition

Our submission for the Trinity Hackathon IX.

## Quick Start
Requires [Rust](https://rustup.rs)
``` shell
$ git clone git@github.com/zemongrabber/trinity-competition
$ cd trinity-competition
$ python3 -m venv .env
$ source .env/bin/activate
(.env) $ pip install maturin
(.env) $ cd env_sim
(.env) $ maturin develop
(.env) $ cd ..
(.env) $ python3 main.py
```
